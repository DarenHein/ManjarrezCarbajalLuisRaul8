
### Temario para Aprender HTML

#### Módulo 1: Introducción a HTML

1. **Introducción a la Web**
   - Definición y conceptos básicos.
   - Estructura cliente-servidor.

2. **HTML: HyperText Markup Language**
   - ¿Qué es HTML?
   - Versiones de HTML.
   - Sintaxis y estructura básica de un documento HTML.

#### Módulo 2: Estructura Básica de HTML

3. **Elementos y Etiquetas**
   - Definición de elementos y etiquetas.
   - Elementos de bloque y en línea.

4. **Documentos HTML Básicos**
   - Declaración DOCTYPE.
   - Etiquetas `<html>`, `<head>`, `<title>`, y `<body>`.

#### Módulo 3: Texto y Enlaces

5. **Manipulación de Texto**
   - Etiquetas de encabezado (`<h1>`, `<h2>`, ...).
   - Párrafos (`<p>`).
   - Etiquetas de formato de texto (`<strong>`, `<em>`, `<u>`).

6. **Listas y Enlaces**
   - Listas ordenadas (`<ol>`).
   - Listas desordenadas (`<ul>`).
   - Listas de definición (`<dl>`).
   - Enlaces (`<a>`).

#### Módulo 4: Imágenes y Multimedia

7. **Inclusión de Imágenes**
   - Etiqueta `<img>` y atributos relacionados.

8. **Multimedia: Audio y Video**
   - Etiquetas `<audio>` y `<video>`.
   - Atributos relacionados.

#### Módulo 5: Formularios

9. **Creación de Formularios**
   - Etiqueta `<form>` y sus atributos.
   - Campos de entrada (`<input>`), áreas de texto (`<textarea>`), y selecciones (`<select>`).

10. **Elementos de Formulario Adicionales**
   - Botones (`<button>`).
   - Etiquetas de agrupación (`<fieldset>` y `<legend>`).
   - Etiqueta `<label>`.

#### Módulo 6: Tablas y Marcos

11. **Tablas en HTML**
    - Estructura de una tabla.
    - Etiquetas `<table>`, `<tr>`, `<td>`, `<th>`.
  
12. **Marcos en HTML (deprecated)**
    - Concepto de marcos y su uso.
    - Etiquetas `<frameset>`, `<frame>`, `<noframes>`.

#### Módulo 7: Semántica y Accesibilidad

13. **Semántica HTML5**
    - Nuevas etiquetas semánticas: `<article>`, `<section>`, `<nav>`, `<header>`, `<footer>`, etc.

14. **Accesibilidad en HTML**
    - Principios básicos.
    - Atributos `alt` y `title`.

#### Módulo 8: CSS y Estilo Básico

15. **Introducción a CSS**
    - ¿Qué es CSS?
    - Vinculación de estilos (interno, externo e incrustado).
  
16. **Propiedades CSS Básicas**
    - Colores, fondos, márgenes, rellenos, bordes, y tipografía.

#### Proyecto Final: Construir una Página Web Básica

17. **Proyecto Final**
    - Aplicar los conocimientos adquiridos para construir una página web básica.
    - Integrar texto, imágenes, enlaces, formularios y estilos CSS.
